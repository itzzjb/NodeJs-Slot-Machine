Initializing the package.json file  -> npm init

Now we can install packages for this project -> npm i "package-name" || npm install "package-name"
> Here we installed promt-sync package
> It is used to get input by the user via console
> This will result in creating package-lock.json file

Running a file -> node "file-name"
> You need to make sure that you are in the same directory as the file
